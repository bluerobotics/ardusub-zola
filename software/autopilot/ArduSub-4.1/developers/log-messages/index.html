Onboard Log Messages

This is a list of log messages which may be present in DataFlash (.bin) logs produced and stored onboard ArduSub vehicles (see Log Parameters for creation details). It is possible to add a new message by modifying the firmware.

DataFlash logs can be downloaded and analysed from a computer or through BlueOS.
